# ReFashion Reward System — Local Development Setup

## Prerequisites

- Node.js 18+
- PostgreSQL running locally
- Git

---

## 1. Clone and install

```bash
# Blockchain
cd refashion-project/blockchain
npm install

# API
cd ../api
npm install
```

---

## 2. Start the local Hardhat blockchain

Open a dedicated terminal and keep it running:

```bash
cd blockchain
npx hardhat node
```

This prints 20 funded test accounts. Copy account #0's private key — you'll need it.

---

## 3. Deploy the contract

In a second terminal:

```bash
cd blockchain
npx hardhat run scripts/deploy.js --network localhost
```

Expected output:
```
✅ ReFashionToken deployed to: 0x5FbDB2315678...
📄 Contract config written to: ../api/src/blockchain/contract.json
✅ Smoke test passed
```

The deploy script automatically writes `api/src/blockchain/contract.json`.
No manual ABI copying needed.

---

## 4. Set up the database

```bash
createdb refashion
psql refashion -f api/src/models/schema.sql
```

---

## 5. Configure the API

```bash
cd api
cp .env.example .env
```

Edit `.env`:
- `MINTER_PRIVATE_KEY` → paste account #0's private key from Hardhat node output
- `ENCRYPTION_KEY` → generate with `openssl rand -base64 32`
- Leave everything else as-is for local dev

---

## 6. Start the API

```bash
cd api
npm run dev
```

---

## 7. Test the reward flow

```bash
# Create a wallet for user "user-123"
curl -X POST http://localhost:3000/wallet/create \
  -H "Content-Type: application/json" \
  -d '{"userId": "user-123"}'

# Issue 50 RFT for a sustainability action
curl -X POST http://localhost:3000/rewards/issue \
  -H "Content-Type: application/json" \
  -d '{"userId": "user-123", "amount": 50, "rewardEventId": "evt-listing-001"}'

# Check balance
curl http://localhost:3000/wallet/user-123/balance

# Deduct 20 RFT
curl -X POST http://localhost:3000/rewards/deduct \
  -H "Content-Type: application/json" \
  -d '{"userId": "user-123", "amount": 20, "rewardEventId": "evt-deduct-001"}'

# Test idempotency (call issue again with same rewardEventId — safe, returns existing tx)
curl -X POST http://localhost:3000/rewards/issue \
  -H "Content-Type: application/json" \
  -d '{"userId": "user-123", "amount": 50, "rewardEventId": "evt-listing-001"}'
```

---

## 8. Run smart contract tests

```bash
cd blockchain
npx hardhat test
```

## Project Structure

```
refashion-project/
├── blockchain/
│   ├── contracts/ReFashionToken.sol   ← ERC-20 contract
│   ├── scripts/deploy.js              ← deploy + smoke test
│   ├── test/ReFashionToken.test.js    ← unit tests
│   └── hardhat.config.js
└── api/
    └── src/
        ├── server.js
        ├── routes/index.js
        ├── controllers/
        │   ├── walletController.js
        │   └── rewardController.js
        ├── services/
        │   ├── walletService.js       ← key encryption/decryption
        │   └── rewardService.js       ← idempotency + DB logging
        ├── blockchain/
        │   ├── client.js              ← ethers.js wrapper
        │   └── contract.json          ← auto-generated by deploy script
        └── models/
            ├── db.js
            └── schema.sql
```
