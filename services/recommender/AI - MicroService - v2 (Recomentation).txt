Integration flow:
Given an input user, 
---> we find from "customer" table, the closest customers ids based on 'age' feature
---> then from "order" table we find the order ids for the corresponding customers ids  
---> then from "order_item" we find the item_id  
---> then from "order_line_item" we find the "product_id" 
---> and finally we output the given input user id paired with the product ids found (recomended products).
