Run locally (Windows PowerShell/CMD):
cd C:\Projects\Refashion\services\order_impact
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --host 127.0.0.1 --port 9200 --reload

Health check: GET http://127.0.0.1:9200/health → {"service":"order-impact","status":"ok"}
This service is localhost-only (bound to 127.0.0.1), so nothing external can hit it.
